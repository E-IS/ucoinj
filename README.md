ucoinj
======

uCoin Java API.


## Compile

Install required dependencies:

 - Install [libsoidum](http://doc.libsodium.org/installation/README.html)
 - Install [Maven 3](http://maven.apache.org/).

Get the source code, then compile using Maven:
 
```bash
$ mvn install
```
